In this project I will use a dataset from [mavenanalytics](https://mavenanalytics.io/data-playground/global-electronics-retailer). Part of this Project will be a Ingestion-Workflow to create a relational database. For this data lake creation, data will also be collected via an API and wbscraping. The data will be cleaned and checked via a Qualitiy Gate. Late we can use it for a deeper analysis of the dataset. The goal is to present skills with SQL, relational Databases and automated workflows.
